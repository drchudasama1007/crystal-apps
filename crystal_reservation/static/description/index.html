Odoo Multi Seller/Vendor Marketplace for your eCommerce Odoo Apps

This Odoo apps module is like Amazon marketplace which allow third party seller or vendors to register on website and sell their products on Odoo and build website marketplace platform where Multi Seller/Vendor Option For Your ODOO E-Commerce webshop allow create their product stores which will convert Your Odoo Store Into An Online Multi Vendor Marketplace. Marketplace supplier and vendor will have option to add stock of their products from the back-end via seller portal access. This odoo apps also provide feature to calculate commission for every seller also it will have separate seller shop page for every seller. Attractive Dashboard is available for all seller where seller can manage everything easily and quickly.

Features

Multi Seller/Vendor Support.

You can create and manage multiple seller/vendor.

Separate Seller Shop Page for Every Seller.

Every seller has different separate seller shop page.

Global Commission for Every Seller.

You can configure global commission for every seller.

Attractive Dashboard for seller and Admin

Seller and admin can see and manage marketplace products, orders, and payments count on the dashboard.

Product Review and Rating.

Review and rating for marketplace products which helps buyer to select right product.

Displayed only Approved Products by Admin.

Marketplace displayed only approved products by admin.

Convert Vendors Site into Complete Advanced Marketplace.

When shop created it will converted to complete advanced marketplace.

Seller Information on Product Page.

User can see particular Vendors informations on selected product page.

Confirm Only Approved Order by Admin.

Seller can confirm only approved sale order by admin.

Seller can Request Multiple Order Payments.

Seller can see his product sales and request payments to admin for multiple orders.
